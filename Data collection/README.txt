This folder contains the files:
    
    load_additional_biological_data.R - R code with functions that load the biological data required for the biological interpretation analysis
    load_evaluation_data.R - R code with functions that load data required for evaluation
    load_model_data.R - R code for parsing data used to generate the input matrices (A, B, G) of the model
    load_reproducibility_data.R - R code with functions for loading the data required for the reproducibility analysis

and the subfolder:

    Data
